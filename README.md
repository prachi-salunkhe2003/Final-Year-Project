LPG Gas Level Detection & Automatic Booking Alert System
🚀 Project Overview

This project is an IoT-based LPG Gas Level Detection and Automatic Booking Alert System designed to monitor gas levels in real-time, detect gas leakage, and automatically notify users via SMS.

The system improves safety, convenience, and efficiency by reducing manual effort in LPG monitoring and booking.

🎯 Objectives
Detect LPG gas level inside the cylinder
Send SMS notifications when gas level is low
Automatically trigger gas booking alert
Detect gas leakage and ensure safety
Provide real-time monitoring
Maintain daily gas usage data

⚙️ System Features
📡 Real-time gas level monitoring
📩 SMS alerts using GSM module
🔥 Gas leakage detection with buzzer alert
🌬 Automatic exhaust fan activation
📊 Data tracking and analysis
🤖 IoT-based smart automation

🧩 Hardware Components
Microcontroller: ATmega328
Voltage Regulator: LM7805
LCD Display: 16x2
Wi-Fi Module: ESP8266
GSM Module: SIM800L
Servo Motor: SG90
Gas Sensor: MQ-6
Load Cell + HX711 (for weight measurement)

💻 Software Used
Arduino IDE – Programming microcontroller
Express PCB – Circuit design and layout

🔄 Working Principle
Load cell measures the weight of LPG cylinder
MQ-6 sensor detects gas leakage
Data is processed using ATmega328 microcontroller
If gas level is low → SMS alert sent to user
In case of leakage →
Buzzer is activated
Exhaust fan turns ON
System enables timely booking and safety alerts

✅ Advantages
⏱ Saves time and effort
🔔 Automatic alerts and notifications
🌍 Remote monitoring capability
🔐 Enhanced safety system
💰 Prevents emergency/black market buying
